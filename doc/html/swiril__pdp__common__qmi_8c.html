QMI network reject cause lookup tables.

Based on table 2.3.2 (UMTS part) of QCOM document 80-VF219-10B, GobiConnectionMgmtAPI.pdf

To use the following Cause Code tables, take the code received from the SDK, perform a range check to determine which table to index, reduce the sdk code by the offset value (Defined constant in front of each table) and use the result as an index into the table. The value at that index is the associated Android RIL error code
